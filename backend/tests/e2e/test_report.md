## 错误日志分析

在测试过程中发现以下错误（已记录但不影响测试通过）：

### 1. Embedding API 错误 (HTTP 404)
```
HTTP Request: POST https://api.deepseek.com/embeddings "HTTP/1.1 404 Not Found"
```
- **原因**: DeepSeek API 不支持 `/embeddings` 端点
- **影响**: 系统自动降级使用 Mock Embedding
- **状态**: 已修复 - 将 embedding 路径改为 10.0.0.3:8000

### 2. MathSolver 执行超时
```
MathSolver Error: Execution timed out.
```
- **原因**: 默认超时时间 5 秒过短
- **影响**: 部分复杂计算可能超时
- **状态**: 已修复 - 将超时时间延长至 10 秒

### 3. 数据库映射错误
```
Failed to create QA record: One or more mappers failed to initialize
```
- **原因**: 存在两个 `Course` 类定义导致映射冲突
- **影响**: 无法保存 QA 记录到数据库
- **状态**: 需手动修复 - 需要统一模型类定义
